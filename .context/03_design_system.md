# Design System

## Brand Identity (Cuby.mx)
- **Primary:** `#0066FF` (Vibrant Tech Blue).
- **Secondary:** `#FFFFFF` (Clean White).
- **Accent:** `#00C853` (Savings Green).
- **Background:** `#F8FAFC` (Slate 50).

## UI Layout (Responsive)
- **Desktop:** Split screen. Left 60% (Calculator + Charts), Right 40% (Chat Interface).
- **Mobile:** Stacked. Calculator on top (collapsible), Chat floating button or bottom sheet.

## Components
- **Chat Bubbles:** distinct colors for User (Gray) vs. Cuby Bot (Blue).
- **Charts:** Use `Recharts` for a simple Bar Chart: [Current Cost] vs [Cost with Cuby].
