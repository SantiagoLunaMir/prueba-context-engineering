# Technology Stack

## Core Framework
- **Next.js 14+ (App Router):** Mandatory for API Routes.
- **Language:** TypeScript.
- **Styling:** TailwindCSS.

## AI Integration
- **SDK:** `Vercel AI SDK` (`npm i ai @ai-sdk/openai` or `google`).
- **Hook:** `useChat({ body: { calculatorState } })`. *Crucial: passing state to the bot.*

## Visualization
- **Charts:** `recharts`.
- **Icons:** `lucide-react`.
